Read me for LFP simulation tool

1. Copy all tool files in the folder to model folder. If not the .mod files of neuronal model in the main directory, you may have to copy lfp.mod to the mechanism directory.

2. Compile the model

3. Load the model in NEURON

4. type : xopen("extracellular_electrode.hoc")  in the terminal to initiate the tool.


5. Set the parameters in the gui. 

6. Run the simulation.
